# Hospital Data Analysis Project

## Project Overview
This project demonstrates **end-to-end data analysis** using **SQL Server** and **Power BI**.  
It simulates a real-world hospital environment to analyze **patient admissions, doctor performance, department workload, and revenue insights**.

- **Objective:** Transform raw hospital data into actionable insights and interactive dashboards.
- **Audience:** Hospital administrators, management, and data analysts.

---

## 🛠 Tools & Technologies
- **SQL Server** – database design, queries, views, stored procedures  
- **Power BI** – interactive dashboards, KPI visualization, reporting  
- **Data Cleaning & Modeling** – handling NULLs, removing duplicates, building relationships
